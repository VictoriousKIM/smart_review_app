# Juso API 신청서 작성 가이드

## API 신청 폼 입력 가이드

현재 프로젝트(Smart Review App)에 맞는 API 신청서 작성 예시입니다.

### 필수 입력 항목

#### 1. API 종류
- ✅ **공개 도로명주소 API** (기본 선택됨)
  - 설명: 속성으로 구성된 공개하는 정보
  - 우리 프로젝트에 적합: 우편번호와 주소 검색에 사용

#### 2. API 유형
- ✅ **검색 API** 선택
  - 설명: REST API 방식으로 주소 검색
  - 우리 프로젝트에 적합: Flutter 앱에서 HTTP 호출로 사용

#### 3. 신청기관 유형
- ✅ **민간기관** 선택
  - 설명: 개인 또는 민간 기업

#### 4. 업체(기관)명
```
스마트 리뷰
```
또는
```
Smart Review
```
- 설명: 프로젝트명 또는 회사명 입력
- 실제 회사명이 있으면 그대로 입력

#### 5. 시스템명
```
Smart Review App
```
또는
```
스마트 리뷰 앱
```
- 설명: 서비스/시스템 이름

#### 6. URL(IP)
**개발 단계:**
```
localhost
```
또는
```
127.0.0.1
```

**운영 단계 (실제 도메인이 있는 경우):**
```
yourdomain.com
```
또는
```
app.yourdomain.com
```

**참고:**
- 개발 단계에서는 `localhost` 또는 `127.0.0.1` 입력 가능
- 운영 도메인이 있으면 실제 도메인 입력
- 여러 도메인 사용 시 "시스템(사이트) 목록" 파일 업로드 사용

#### 7. 시스템 개요 (선택)
```
리뷰 캠페인 플랫폼으로, 사용자가 주소를 입력할 때 우편번호 찾기 기능을 제공합니다.
주소 검색 및 우편번호 조회를 위해 도로명주소 API를 활용합니다.
```
- 설명: 시스템의 목적과 API 사용 목적 간단히 설명

#### 8. 서비스망
- ✅ **인터넷 망** (기본 선택됨)
  - 설명: 일반적인 웹/앱 서비스

#### 9. 서비스 용도
**개발 단계:**
- ✅ **개발(본인인증없이 발급)** 선택
  - 기간: 7일, 30일, 90일 중 선택 (개발 기간에 맞게)
  - 설명: 개발 및 테스트용으로 빠르게 발급 가능

**운영 단계:**
- ✅ **운영** 선택
  - 설명: 실제 서비스 운영용 (본인인증 필요)

#### 10. CSS 파일 경로 (선택)
```
http://business.juso.go.kr/css/addrlinkSample.css
```
- 설명: 기본값 사용 또는 커스텀 CSS 경로 입력
- 우리 프로젝트는 Flutter 앱이므로 기본값 사용 권장

## 여러 시스템 연계 신청

여러 도메인(개발/스테이징/운영)을 사용하는 경우:

1. **"양식 다운로드"** 클릭
2. 엑셀 파일에 도메인 목록 작성:
   ```
   시스템명          | URL(IP)
   Smart Review App  | localhost
   Smart Review App  | staging.yourdomain.com
   Smart Review App  | yourdomain.com
   ```
3. 파일 업로드하여 일괄 신청

## 주의사항

1. **개발용 신청**
   - "개발(본인인증없이 발급)" 선택 시 빠르게 발급 가능
   - 기간 제한이 있으므로 운영 전에 운영용으로 재신청 필요

2. **운영용 신청**
   - "운영" 선택 시 본인인증 필요
   - 승인까지 1-2일 소요
   - 일일 호출 제한 설정 가능

3. **URL 입력**
   - 개발 단계: `localhost` 또는 `127.0.0.1` 가능
   - 운영 단계: 실제 도메인 필수
   - 여러 도메인은 파일 업로드로 일괄 신청

4. **API 유형**
   - **검색 API**: 우리가 구현한 방식 (REST API)
   - 팝업 API: 웹 전용 (우리 프로젝트에는 부적합)

---

## 신청 후 절차

1. **신청 제출**
   - "신청하기" 버튼 클릭
   - 신청 완료 메시지 확인

2. **승인 대기**
   - 개발용: 즉시 또는 몇 시간 내 발급
   - 운영용: 1-2일 소요

3. **API 키 확인**
   - "신청내역" 메뉴에서 확인
   - 승인 완료 시 API 키(confmKey) 표시

4. **프로젝트 적용**
   - `lib/services/juso_api_service.dart` 파일 열기
   - `_apiKey` 값에 발급받은 키 입력
